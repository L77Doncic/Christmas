### 三种形态专属特色

#### 🎄 经典圣诞树

传统绿松枝叶搭配鲜红蝴蝶结、金黄铃铛，粒子缓慢旋转，还原圣诞树最经典的温暖模样，适合营造温馨节日氛围。

#### 🌌 银河圣诞树

蓝紫渐变粒子如星空闪烁，部分粒子化作"流星"流动，顶部星星泛着冰晶光泽，科幻与浪漫的完美结合。

#### 🎁 喜庆礼物盒

正红礼盒搭配耀眼金丝带，顶部超大蝴蝶结格外吸睛，粒子排列成规整盒面，满满都是拆礼物的期待感。

## 🎨 自定义小技巧（进阶玩法）

如果想打造专属圣诞风格，可修改代码中这些参数（无需专业开发经验）：

- **调整颜色**：找到 `classicColor`（经典树颜色）、`galaxyColor`（银河树颜色）等变量，替换为喜欢的色值（如 "#FF6B6B" 是可爱粉红包）
- **修改粒子数量**：调整 `PARTICLE_COUNT` 数值，越大粒子越密集（建议范围：1000-5000）
- **更改旋转速度**：在 `targetRotationY +=` 处修改数值，越大旋转越快
- **更换标题字体**：替换 `Mountains of Christmas` 为其他谷歌字体（如 "Christmas Spirit"）

## ❓ 常见问题

- **Q：移动端点击没反应？** A：确保触屏时点击屏幕中央区域，部分手机需关闭浏览器"省流模式"
- **Q：动画卡顿怎么办？** A：减少粒子数量（修改PARTICLE_COUNT），或关闭浏览器其他占用性能的标签页
- **Q：能否用作圣诞贺卡？** A：完全可以！修改 `drawBottomText` 中的文字，加上你的专属祝福送给朋友
